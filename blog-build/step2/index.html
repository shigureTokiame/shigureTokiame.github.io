# 安装 Shoka 依赖插件

以下插件表格出处是 Shoka 主题作者写的文章 点我访问。

插件名称npm 地址功能依赖程度
hexo-renderer-multi-markdown-it链接md 文件渲染器，压缩 css/js/html必需
hexo-autoprefixer链接给生成的 css 文件们添加浏览器前缀必需
hexo-algoliasearch链接站内搜索功能搜索按钮失灵
hexo-symbols-count-time链接文章或站点字数及阅读时间统计没有统计
hexo-feed链接生成 feed 文件feed 文件没有

如果没有以上插件，那么 Shoka 主题可能会报错 & 无法正确显示 & 部分功能失效。
hexo-renderer-multi-markdown-it 请注意务必升级到最新版本，目前是 0.1.5 版本
