# 使用 Shoka 主题

打开根目录的 _config.yml （配置文件）

在配置文件中里面找到 theme ，并将 Hexo 的主题从默认的 landscape 改为 shoka （顺便可以删除根目录的 _config.landscape.yml ）

# 配置插件

  1. 之后在根目录下的配置文件 <root>/_config.yml 中添加 markdown 设置，主要用于渲染文章。

  2. 在根目录的配置文件 <root>/_config.yml 中添加 minify 配置，主要用于压缩 css/js/html 文件。

  3. 必须停用代码高亮功能，不然会导致文章中的代码块部分无法正常显示，在根目录的配置文件 <root>/_config.yml 中，找到 highlight 和 prismjs ，将 enable 改成 false 。

  4. 在根目录的配置文件 <root>/_config.yml 中添加 autoprefixer 配置，用于自动添加 css 前缀以兼容不同浏览器。

  5. 在根目录的配置文件 <root>/_config.yml 中添加 algolia 配置，用于文章搜索功能。
    如果不知道 appId、apiKey 和 adminApiKey 如何获取，后面会讲到的。

  6. 在根目录的配置文件 <root>/_config.yml 中添加 feed 配置

再次使用 hexo clean ， hexo g 和 hexo s ，效果如下图所示：

# 特殊配置

# Algolia 搜索插件基础配置

  • 首先到 algolia 官网，注册一个账号

  • 登录后创建一个 index，顺便可以将其命名为 Shoka

  • 在左上角的 Application 处，创建一个名为 Shoka 的 Application

  • 然后选择地域（随意选择）

  • 然后打开 Setting --> Team and Access --> API Key

  • 将里面的 Application ID 、 search-only ApiKey 、 Admin API Key 分别复制并放入 algolia 中

# Valine 评论插件基本配置

  • 首先到 LeanCloud 官网，注册一个账号

  • 在华北或者华东创建一个名为 Shoka 的应用

  • 进入应用，选择数据存储 --> 结构化数据 --> 创建 Class，分别创建 Comment 和 Counter 两个 Class，分别存储评论和阅读量数据，只需添加 Class 名称即可，其他的可以默认，创建后如下图所示。

  • 然后打开设置 --> 应用凭证，将其中的 AppID 和 AppKey 复制粘贴到主题基础配置文件 (_config.shoka.yml) 中的 Vallain 插件中

  • 然后重新再次使用 hexo clean 、 hexo g 和 hexo s 后刷新网页，就能够在任意一篇文章的最底下看到评论区了。

# Valine 评论插件测试

  • 用自己的邮箱和名称，去发布一条评论，随后打开 LeanCloud 官网中，回到数据存储 --> 结构化数据 --> Comment 后，刷新页面，就可以看到评论结构已经出现了。

  • 将 mailMd5 中的值复制到 valine 配置中的 master 下方，这样在我们自己就能够在评论区显示 主人 了。

# Valine 评论插件安全配置

  • 在 Comment 中，找到 权限 ，修改 Class访问权限 和 列权限 。

# Gravatar 配置评论头像

  • 打开 Gravatar 的官网，通过右上角使用刚才在 评论区 使用的邮箱进行注册。

  • 并在邮箱内找到验证信息，并成功注册。

  • 新用户可以直接在这个界面上传头像，也可以点击下面暂时跳过，之后再上传头像。

  • 如果跳过了，则可以在这两个位置上传，并在页面左侧上传头像。

  • 在上传完头像后，一般需要等待 5-10 分钟即可生效，但是某些人可能会出现不可抗力原因 比如我 ，需要等待几天才会生效。
  • 之后打开 Gravatar 的 Email Checker，将自己的邮箱输入进去来获得头像的链接

  • Avatar URL 就是头像的链接，比如。
    https://secure.gravatar.com/avatar/a7e2c6d017925b9b3f129101f37a6b18e3f02285728299f81bef28960147d377
  • 因为 Gravatar 进行了更新，所以导致这边的 URL 也同样进行了更新 （我说怎么链接看着不对劲） 。
  • 之后在评论那里将连接放入，便可以显示头像了，前提是等待几分钟 （某些倒霉蛋甚至需要几天） 。
