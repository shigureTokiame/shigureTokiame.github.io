从零开始的博客搭建教程

# 前言

  • 这个博客是依靠 Hexo 框架进行搭建的。
  • 博客主题是 Shoka，作者是 霜月琉璃，一位兼具计算机与医学的大佬。
  • 左下角的看板娘是由 GardenHamster 的魔改 Live2d-widget，叫做 Live2d-web。
  • 建议在搭建博客之前看一下这些文档。

在搭建博客之前，强烈建议先拥有一个 Github 账号。

警告，该博客搭建虽然面向小白，但仍一定的计算机基础。

# 开发环境

  • 使用的编译器是 Visual Studio Code（VS Code），VS Code 是不自带中文的，中文插件能够在 VS Code 里面找到，插件名叫做： Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code 。
  • 最新版本的 Nodejs，安装的时候一定要勾选 Add to PATH 选项（默认是勾选的）。
  • 最新版本的 Git（Git Bash）。

在 Hexo 博客所在的文件夹中打开 Git Bash，并输入下方任意一个指令（考虑到我的个人问题，下面提供两种方法）。

在 Git Bash 中出现 Resolving deltas 100% done 的时候，也就表明安装成功了（若在安装期间遭遇问题，这里并不会写解决方法，还请各自百度:）

接着在文件夹中打开 themes，里面会多出一个名为 shoka 的文件夹，在 VS Code 中如图所示

其中的 example 为实例文件， _config.yml 是主题配置文件（很重要）

  1. 一般不建议直接修改这个主题文件夹 shoka 里面的 _config.yml 。
  2. 最好在根目录中新建一个 _config.shoka.yml 文件，然后将 /themes/shoka/ 里面的 _config.yml 中的内容全部复制到 _config.shoka.yml 中进行修改，反正最终 hexo generate 的时候都会将这两个文件合并。
  3. 包括 shoka 里面自带的图片，最好也在根目录里面的 /source/data/images 中，放入同名文件来进行替换，什么？你告诉我你没有 data 文件夹？那就自己创建一个吧，images 文件夹同理。
    以头像为例，shoka 中头像的位置是 /themes/shoka/source/images/avatar.jpg ，那么要替换默认头像，就需要在 /source/_data/images 中，将一个图片名称同样改为 avatar.jpg 后，就可以对默认头像进行替换了。
