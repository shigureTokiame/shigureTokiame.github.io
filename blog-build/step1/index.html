# 前言

  • 这个博客是依靠 Hexo 框架进行搭建的。

  • 这个博客的主题是 Shoka，作者是霜月琉璃，一位兼具计算机与医学的大佬。

  • 左下角的看板娘是由 GardenHamster 魔改的 Live2d-widget，叫做 Live2d-web。

  • 建议在搭建博客之前看一下这些文档

警告，该博客搭建虽然面向小白，但仍一定的计算机基础

# 开发环境

  • 使用的编译器是 Visual Studio Code（VS Code），VS Code 是不自带中文的，中文插件能够在 VS Code 里面找到，插件名叫做： Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code
  • 最新版本的 Nodejs，安装的时候一定要勾选 Add to PATH 选项（默认是勾选的）
  • 最新版本的 Git
