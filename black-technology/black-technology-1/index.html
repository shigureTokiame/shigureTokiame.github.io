# 前言

之前玩游戏的时候（黄油:），出现过 正常运行 和 以管理员身份运行 都无法正常打开游戏的情况，甚至当时将系统地区调成了对应地区也无法打开，于是当即就去网上找了好长时间。
然后在网上冲浪找解决办法的时候，找到了这么个插件 ——Locale Emulator

# 说明

其实这个问题主要是国内电脑的普遍问题，以日本游戏举例 （典例），部分日本游戏（或者文件）可能会出现乱码、运行不了、闪退、显示不正常，文字错误等等一系列问题。而这种插件可以解决这些问题，统称为转区工具。

# 乱码原理

  • 非 Unicode 应用程序的语言应与系统区域所定义语言的脚本或类型不相同，从而导致在应用程序的用户界面中显示乱码。

    简单说明就是（人话:），电脑的 Unicode 跟软件或游戏的不匹配，导致游戏出现各种问题。

# 转区原理

  • 在基于 Unicode 的平台中模拟运行非 Unicode 应用程序所需的语言环境，使用通常称作 系统区域 的系统范围变量（或适用于非 Unicode 应用程序的语言）将应用程序的非 Unicode 文本数据在内部转换为 Unicode。

    简单说明就是（人话:），转区工具会将电脑的 Unicode 转换成软件或游戏所对应的 Unicode，从而顺利解决这一系列问题。

# 食用说明

  1. 下载完之后打开压缩包 （我拒绝回答一切跟文件夹或压缩包相关的问题）。
  2. 在随便哪个地方创建一个文件夹，然后将压缩包内所有的文件放入。
  3. 然后点击 LEInstaller.exe ，开始安装。
  4. 选择安装方式，三种方法都可以，计算机小白建议直接选择第一个。
  5. 安装完成后跳出下面这个框就算是安装成功了。
  6. 右键文件后出现 Locale Emulator 后即可正常食用。

进阶功能可以打开 LEGUI.exe 进行修改和查看。
