# 前言

作为一名比较菜的程序员，时常会因为打开软件花费一些时间，可能会因为没有放快捷方式的习惯，经常会去找对应的文件夹来启动文件，这导致花费了我异常多的时间。然后在网上找资料的时候，看到了 uTools 的介绍，感觉还不错，于是就去用了，用了之后呢，只想说超级好用，现在已经离不开了。（uTools 太强辣！）

# 功能介绍

uTools 的功能，我感觉大概可以分成 两个大类 ，快速启动软件功能和即时好用的插件功能。

# 快速启动

  • 这个快速启动功能简直让我眼前一亮，输入关键词就可以快速打开电脑中的软件（跟搜索栏很像对吧？我也觉得很像，但是比搜索栏更好用），我有了解过，MacOS 有这种快速启动的功能 （但是我没有用过 Mac 系统） ，使用 alt+空格 就会跳出一个搜索栏，输入关键词之后，软件也会在搜索栏出现，接着就可以直接打开了，听起来也挺快的（至少比去文件夹里面找快很多）。

    这个功能可能说的有点笼统，因为 Windows 系统上面也有搜索栏，但是我不太喜欢用，原因是搜索有延迟，而且很多软件不能直接靠关键词搜索到 （这样真的很不爽） 。

# 插件功能

  • 这个功能也是重中之重，可以在 插件应用市场 里面下载超级多的插件（应有尽有！），本身自带的插件也是非常好用。
    取色、翻译、批量修改文件名、Markdown、识别图中文字 等等一系列好用的插件，全部都集成在了 uTools 中，方便管理且可以随时调用这些插件，完全不影响使用。

# 食用方法

  1. 先去 uTools 的官网下载 uTools，官方网址是：https://www.u.tools，然后根据自身电脑来下载对应的 uTools。
  2. 双击打开安装文件。
  3. 安装完后会在工具栏出现 uTools 图标，使用默认按键 Alt+Space（空格） 即可打开 uTools 菜单。
  4. 右边的小黑球就是详细菜单，打开后如下图所示。

在个人中心的设置里面可以按照自己习惯爱好修改 uTools 的功能和按键

# 快速打开软件功能食用方法

  • 直接通过 Alt+Space（空格） 打开 uTools 菜单后，直接输入软件的关键词，便会跳出相关软件，按下回车即可打开列表中的第一个应用。
  • 可以在 uTools 菜单中打开右边 uTools 的默认图标。

    在 个人中心 ——> 本地文件启动 中，可以手动添加启动文件（如果 uTools 通过关键词搜索不到的话)

什么都可以通过这种快速打开软件的方式启动，原理就是隔空双击打开软件（例如：Excel/PPT/WORD，游戏，图片等等）

# 插件功能食用方法

  • 通过 Alt+Space（空格） 打开 uTools 菜单后，在右边的 uTools 默认图标处进行点击，就会自动打开插件应用市场，插件应用市场中可以下载各式各样的插件，下载完后能立刻使用。
  • 在已安装插件中能够看到自己目前所拥有的插件。
  • 最后在 uTools 菜单中，直接输入插件的关键词或者名字，就可以直接使用。

# 其余特殊功能

  • 全局快捷键：通过快捷键能够快速的打字使用，效果更佳，当按下快捷键的时候，uTools 会自动打开并填充搜索栏，做到更快更方便的打开软件。
  • 数据存储：没什么主要用处，主要用于 uTools 的 插件 与 本地文件启动 中，在右侧能够查看所有关于 uTools 的 json 文件（本地启动，插件，版本等等）
  • 在打开 uTools 详细菜单后，再次点击 uTools 的图标后能够出现一部分设置和选项。
