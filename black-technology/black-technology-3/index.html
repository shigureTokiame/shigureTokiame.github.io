# 前言

其实我很早就打算写关于这个 Internet Download Manager 了，但是因为各种原因不打算写 因为正版的 IDM 需要花钱购买会员 。虽然话是这么说，但是我还是写了。(有条件的还是支持一下正版吧)

但是在开始之前，需要先了解功能，并做好前面的准备工作

# 功能介绍

IDM 作为一个浏览器的下载器，其最大的作用就是十分直观的增强下载速度，至少可以提升 3-5 倍的下载速度，如果网速允许，甚至可以更高；真正意义上的下载界的天花板

当然，除了下载速度快，还自带一些别的功能，例如：

  • 病毒扫描功能

    保护用户被恶意软件或木马病毒骚扰，从而集成了 自动查毒 的功能的 IDM，在其中选中 杀毒软件 后，每当下载完成，杀毒软件就会自动进行扫描

  • 科学下载环境

    众所周知，外网下载，懂的都懂，总之就是外网的东西在配置完端口跟代理之后，下载也能被提速

  • 自动嗅探资源
    自动在视频、图片、文字资源附近，增加一个 下载该 XXX

  • 爬取网页下载

    选择爬取的网页，并将所有能够下载的内容全部显示，适合用来批量下载图库、视频、文章等

    以及其他功能 我也没有使用过全部功能，所以我也不知道

# 准备工作

  1. 能够下载插件的浏览器（谷歌，火狐，360，Edge 等）

  2. IDM（这才是本体）

    下载链接：https://www.lanzoub.com/iCWxz1uwprvc 访问密码：sgr9

# 食用方法

  1. 首先从三件套中解压 IDM 压缩包

  2. 打开 Bin ，右键 !)绿化.bat ，选择管理员打开 不用管理员打开会导致无法正常使用

  3. 然后打开 IDMan ，会显示如下界面

  4. 选择 选项 ，打开后，在 常规设置 中勾选如下两个按钮，勾选之后点击 确认

  5. 随后打开 浏览器 （以下载链接的谷歌浏览器作为演示），打开 扩展程序 ，能够看到在绿化完之后， 扩展程序 自动添加了 Internet Download Manager ，找到后勾选

  6. 下载完之后，使用任意下载链接进行测试，如出现以下界面，则表明已经完成了所有的步骤，能够正常使用了

  7. Enjoy

其余浏览器也会因为绿化后，于 扩展程序 中同样会出现 Internet Download Manager ，所以其他浏览器也可以支持使用 IDM 进行下载
